# metasploit-alpine
Metasploit Framework Installation In Alpine Linux

<br>

```bash
apk add sudo curl
curl -LO https://raw.githubusercontent.com/ghoste9624/metasploit-alpine/refs/heads/main/install.sh 
chmod +x install.sh
./install.sh
```
